Tentang Saya



Nama saya Zefnath C.H Nendissa biasa dipanggil Zefnath atau Zefth. Saya lahir di Ambon, 22 Maret 2001, saya anak pertama dari 3 bersaudara, ayah saya bernama Junus Nendissa dan ibu saya bernama Marice Sahanaya. Ayah bekerja sebagai Buru dan ibu saya sebagai PNS.
Saya bersekolah SD di SD Negeri Teladan Ambon, saya melanjutkan SMP di SMP Negeri 19 Ambon. Saya melanjutkan SMK di SMK Negeri 7 Kota Ambon, disitulah saya mengenakan seragam putih abu-abu dan menuntut ilmu selama 3 tahun. Saya mengambil jurusan Multimedia saat SMK, dan melanjutkan pendidikan di Universitas Teknologi Digital Indonesia dengan Fakultas Teknologi Informasi jurusan Informatika di kota Yogyakarta.
Semenjak duduk dibangku SMK saya mengikuti banyak kegiatan yang bertema multimedia seperti cameramen ataupun film, tidak lupa juga saya sering aktif di bidang seni rupa seperti cipta baca puisi, nyanyi maupun theater.
Memasuki bangku perkuliahan saya mengikuti beberapa organisasi diantaranya IKPM Ambon-Yogyakarta periode 2021-2022, disana saya menduduki jabatan sebagai anggota divisi kominfo, didalam lingkup kampus saya mengikuti beberapa organisasi diantaranya PMK EXODUS UTDI dan DPM UTDI. Di DPM sendiri saya menduduki jabatan anggota devisi hubungan dalam negeri selama 1 priode (2021-2022)
